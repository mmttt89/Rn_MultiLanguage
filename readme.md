## Explanation
This is a simple react native app with foucs on DarkMode(not device darkmode) and Multilanguage feature. 

### Installation
- Run **`yarn`** or **`npm i`**

extra step for ios:
- cd ios then **`Pod Install`**

### Run
  - android: **`yarn android`** or **`npm android`**
  - ios: **`yarn ios`** or **`npm ios`**
